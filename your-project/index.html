The goal is to make this page visible via tor hidden services. We are using nginx to serve tor hidden services with docker-compose to make it all happen.

TOR Service Composer

Use this compose setup to generate a tor hidden service with nginx serving static assets. Simply replace the contents of the folder testproject with your static assests. Your index.html file should be located in tor-composer/your-project/index.html.


Once you have added all your files, in the terminal, at the root of your project issue the following command
docker-compose up --build
Once you see it built and both containers are running, you can use the command following code to enter the container.
docker exec -it hiddenservices /bin/sh
Once in the container you can issue the command
cat hidden-services/hostname
This will print your .onion address to the terminal. Open tor browser and navigate to your new hosted site. It would be a good idea to backup your docker volume to avoid losing your hidden-service keys and address.
